# WinFormsTicTacToe

A TicTacToe game built in C# with a WinForms GUI — companion to the 
console version in [TicTacToe](https://github.com/KacperZys/TicTacToe).

## Features
- Two-player game on a single machine
- Detects wins and draws automatically
- Reset button to start a new game without restarting the app

## What I learned
Building the same game in both console and GUI form showed me how 
separating game logic from the interface matters — the core rules stayed 
the same while the presentation layer changed completely.

## How to run
1. Clone the repo
2. Open in Visual Studio
3. Build and run — no extra dependencies needed

## Tech
